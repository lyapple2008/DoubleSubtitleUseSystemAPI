* 识别结果 **持续增长**
* 中间会 **回退 / 修改**（例如 “级别了” → “级别的”）
* `isFinal=false` 表示 **当前句子还没结束**
* 可能突然 **重启新句子**（最后 `"尤其是"`）

如果你要做 **实时字幕 + 翻译队列**，目标其实是：

1. **尽早翻译**（低延迟）
2. **避免抖动**（不要反复修改）
3. **保证语义完整**

所以核心问题是：

> 如何从不断变化的 partial result 中找出 **稳定片段 (stable segment)**，加入翻译队列。

---

# 一、核心思路（稳定前缀算法）

利用一个非常重要的性质：

**新的识别结果往往是旧结果的前缀扩展或轻微修改**

例如：

```
A: 你的问题已经
B: 你的问题已经根本
C: 你的问题已经根本就不是
```

我们可以做：

```
stable_prefix = LCP(previous, current)
```

即：

**最长公共前缀（Longest Common Prefix）**

因为：

> 如果一段文字在多个连续结果中保持一致，它基本不会再改变。

---

# 二、断句策略设计（推荐方案）

结合 **稳定前缀 + 标点 + 长度 + 停顿**

设计四层判断：

```
                新识别结果
                      │
                      ▼
            计算稳定前缀 LCP
                      │
                      ▼
        判断是否满足“可翻译条件”
      ┌───────────┬────────────┬────────────┐
      │ 标点断句   │ 长度阈值    │ 停顿时间     │
      └───────────┴────────────┴────────────┘
                      │
                      ▼
               加入翻译队列
```

---

# 三、具体算法

维护三个变量：

```
lastResult        上一次识别结果
committedText     已加入翻译队列的文本
pendingText       当前未提交部分
```

每次收到新结果：

```
currentResult
```

执行：

---

## Step1 计算稳定前缀

```python
def lcp(a, b):
    i = 0
    while i < min(len(a), len(b)) and a[i] == b[i]:
        i += 1
    return a[:i]
```

```
stable = LCP(lastResult, currentResult)
```

---

## Step2 找出新增稳定部分

```
newStable = stable[len(committedText):]
```

即：

```
已稳定文本 = stable - 已提交部分
```

---

## Step3 判断是否断句

满足以下任一条件：

### 条件1：标点断句（最优）

```
。 ， ？ ！ ；
```

例子：

```
你的问题已经根本就不是一个级别的，
```

断句：

```
你的问题已经根本就不是一个级别的，
```

---

### 条件2：长度阈值

例如：

```
> 12 字
```

防止长句不输出。

---

### 条件3：停顿时间

如果：

```
500ms 没有变化
```

认为一句结束。

---

# 四、加入翻译队列

伪代码：

```python
stable = LCP(lastResult, currentResult)

newStable = stable[len(committedText):]

sentence = extractSentence(newStable)

if sentence:
    translationQueue.push(sentence)
    committedText += sentence

lastResult = currentResult
```

---

# 五、extractSentence 规则

推荐优先级：

### 1 标点断句

```
regex = r".*?[，。！？]"
```

例：

```
你的问题已经根本就不是一个级别的，
```

输出：

```
你的问题已经根本就不是一个级别的，
```

---

### 2 长度断句

例如：

```
maxLen = 15
```

```
你的问题已经根本就不是一个级别的
```

截断：

```
你的问题已经根本就不是一个级别的
```

---

### 3 停顿断句

当：

```
deltaTime > 600ms
```

输出 pending。

# 二、断句器实现

文件：

```
SpeechSentenceSegmenter.swift
```

核心逻辑：

```swift
import Foundation

class SpeechSentenceSegmenter {
    
    private var lastResult: String = ""
    private var committedText: String = ""
    
    private var lastUpdateTime: Date = Date()
    
    private let maxSentenceLength = 18
    private let pauseThreshold: TimeInterval = 0.7
    
    var onSentenceReady: ((SentenceSegment) -> Void)?
    
    func processResult(_ text: String) {
        
        let stable = longestCommonPrefix(lastResult, text)
        
        let newStable = String(stable.dropFirst(committedText.count))
        
        if let sentence = extractSentence(from: newStable) {
            
            committedText += sentence
            
            let segment = SentenceSegment(
                id: UUID(),
                text: sentence,
                timestamp: Date()
            )
            
            onSentenceReady?(segment)
        }
        
        lastResult = text
        lastUpdateTime = Date()
    }
    
    func checkPause() {
        
        let now = Date()
        
        if now.timeIntervalSince(lastUpdateTime) > pauseThreshold {
            
            let remaining = String(lastResult.dropFirst(committedText.count))
            
            if remaining.count > 0 {
                
                let segment = SentenceSegment(
                    id: UUID(),
                    text: remaining,
                    timestamp: Date()
                )
                
                committedText += remaining
                
                onSentenceReady?(segment)
            }
        }
    }
}
```

# 三、最长公共前缀

```swift
extension SpeechSentenceSegmenter {
    
    private func longestCommonPrefix(_ a: String, _ b: String) -> String {
        
        let minLength = min(a.count, b.count)
        
        var index = 0
        
        let aChars = Array(a)
        let bChars = Array(b)
        
        while index < minLength && aChars[index] == bChars[index] {
            index += 1
        }
        
        return String(aChars[0..<index])
    }
}
```

这个算法的作用是：

```
previous: 你的问题已经根本
current:  你的问题已经根本就不是
```

得到：

```
稳定部分 = 你的问题已经根本
```

# 四、断句规则

```swift
extension SpeechSentenceSegmenter {
    
    private func extractSentence(from text: String) -> String? {
        
        if text.isEmpty {
            return nil
        }
        
        if let range = text.range(of: "[，。！？]", options: .regularExpression) {
            
            let sentence = String(text[..<range.upperBound])
            
            return sentence
        }
        
        if text.count >= maxSentenceLength {
            
            return String(text.prefix(maxSentenceLength))
        }
        
        return nil
    }
}
```

规则优先级：

1️⃣ 标点断句
2️⃣ 长度断句
3️⃣ 停顿断句
